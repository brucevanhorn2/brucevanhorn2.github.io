Highly Experienced

I have over 30 years of industry experience. I've written everything from military flight simulators to desktop, mobile, and web apps. For the last 12 years, I've been working on Visual One Intelligence as the principal software engineer.

Visual One Intelligence is an analytics application designed to ingest data from corporate IT infrastructure. Through an extract-transform-load process these ingested data move through a data pipeline making their way to several databases and a data warehouse.

The overall system is implemented as a set of microservices hosted in Azure. The Azure environment is managed using Terraform and Ansible.

The front-end was created using React, ag-grid, and Highcharts. The application features AI components from Azure and IBM WatsonX.

Naturally, I am hand-on and well versed with this entire array of technology.

In addition to the technical stuff, I also manage the development staff and the department budget. I'm responsible for hiring team members and developing their skills.
